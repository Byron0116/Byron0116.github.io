API简介

API(Application Programming Interface):应用程序编程接口
Java API：JDK中提供的各种功能的Java类。可根据JavaAPI帮助文档查看详情。

String

该类位于java.lang包下,Java中的所有字符串文字(如”abc”)都为此类的对象.

字符串对象一旦创建,其 内容(字符序列) 不能被更改.

创建String对象的两种方式

  1. 直接赋值(常用且内存占用低)
  2. 构造方法
Method Description
public String() 创建空白字符串
public String(String original) 根据传入的字符串,创建字符串对象
public String(char[] value) 根据传入的字符数组,创建字符串对象
public String(byte[] bytes) 根据传入的字节数组,创建字符串对象

String内存模型

StringTable(字符串常量池):JVM在JDK7之后,在堆内存中分配的一块区域,用于存放常用的字符串.

  • 直接赋值

    当使用双引号直接赋值时,系统会检查该字符串在串池中是否存在。
    若不存在,则创建一个新的字符串;否则复用字符串。

  • 构造方法

    此方式创建的字符串不会复用

    综上，使用直接赋值的方式更节约内存。

String的常用方法

字符串比较

  • 关于==号比较
    基本数据类型:比的是数据值
    引用数据类型:比的是地址值
    字符串是引用数据类型，因此比较的是地址值。
  • 关于比较方法
    比较的是内容
Method Description
boolean equals(Object anObject) 仅当anObject不为null且内容完全相同才返回true
boolean equalsIgnoreCase(String anotherString) 将字符串与字符串比较，忽略大小写

==比较的结果为: false
equals比较的结果为: true
equalsIgnoreCase比较的结果为: true
