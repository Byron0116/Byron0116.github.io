文件(File)

Java文件类是文件和目录路径名的抽象表示(‘An abstract representation of file and directory pathnames.’)。
该类主要用于文件和目录的创建、文件的查找和文件的删除等,未涉及文件的读写。关于文件的读写将在下节IO流中介绍。
File对象用路径来表示。

  • 磁盘中实际存在的文件和目录(文件夹)需要用File对象表示，但是File对象代表的可能不是一个真实存在的文件或目录。

File类的构造方法

Method Description
public File(String pathname) 根据文件路径创建文件对象
public File(String parent, String child) 根据父路径名的字符串和子路径名的字符串创建文件对象
public File(File parent, String child) 根据父路径对应文件对象和子路径名的字符串创建文件对象

文件2: D:\Desktop\test1.txt

创建、删除

Method Description
public boolean createNewFile() 创建一个新的空文件
public boolean mkdir() 创建单级文件夹
public boolean mkdirs() 创建多级文件夹
public boolean delete() 删除文件、空文件夹
  • 关于createNewFile方法：
    1. 使用createNewFile方法时,会提示异常(IOException)。
    2. 如果文件路径不存在则能够成功创建(true)，反之创建失败(false)。
    3. 当成功创建文件之后，再执行一次createNewFile会返回false。
    4. 如果父级路径不存在，则方法会出现异常IOException。
    5. 方法创建的一定是文件，如果路径中不包含后缀名，则创建一个没有后缀的文件。
  • 关于mkdir方法
    Windows中的路径是唯一的，如果当前路径已经存在，则创建失败(false)。
  • 关于mkdirs方法
    1. 既可以创建多级文件夹，也可以创建单级文件夹。

    1. delete方法默认只能删除文件和空文件夹,且是直接删除,不经过回收站。
    2. 若删除的是有内容的文件夹则删除失败(false)。

Method Description
public File[] listFiles() 获取当前路径下的所有内容(文件+文件夹)

    1. 获取当前文件夹下的所有内容(包括隐藏文件夹),把所有内容的路径放到File类型的数组中返回.

IO流

数据流：是指在计算机的输入输出之间运动的数据序列。
输入输出是相对程序来说的，程序扮演两个角色：源和目的。
IO流是存储和读取数据的解决方案。用于读写文件中的数据(数据可以在文件中或网络中)。

  • 按流的方向分为
    1. 输入流(InputStream)
      文件➡程序
    2. 输出流(OutputStream)
      程序➡文件
  • 按操作文件类型分为
    1. 字节流
      可以操作所有类型的文件。
    2. 字符流
      只能操作纯文本文件。

IO流体系

字节流

FileOutputStream

可以将程序中的数据写到本地文件中，是字节流的基本流。

  • FileOutputStream常用方法
Method Description
public void write(int b) 将指定的整型数据b的低字节写入输出流
public void write(byte[] b) 把字节数组b中的数据写入输出流
public void write(byte[] b, int off, int len) 把字节数组b中从索引off开始的len个字节写入输出流
public void close() 释放资源
  • 实现步骤
    1. 创建FileOutputStream对象(在程序和文件直接建立数据通信通道)
      参数是字符串表示的路径或者是File对象。
      如果文件不存在会创建一个新文件，但是要保证父级路径存在。
      如果文件已存在，则会清空文件。
    2. 写出数据(在通道中传输数据)
    3. 释放资源(关闭通道)
      每次使用完流后要释放资源
  • 换行
    使用转义字符'\r\n'进行换行。也可简写其中一个。
  • 续写
    上述使用的构造方法都是单参构造，只需给出路径即可，但FileOutputStream的重载构造方法中有两参构造。
    如public FileOutputStream(String name, boolean append)其中append默认值为false，表示创建对象时不进行续写(即默认清除文件内容)；在创建对象时增加第二个参数为true即可实现续写。

FileInputStream

可以将本地文件中的数据读取到程序中。

  • FileInputStream常用方法
Method Description
public int read() 从输入流中读取一个字节，返回字节值
public int read(byte[] b) 从输入流中读取，写入b。返回实际读取字节数
public int read(byte[] b, int off, int len) 从输入流中读取len个字节，从索引off开始写入b。返回读取字节数
  • 实现步骤
    1. 创建FileInputStream对象
      如果文件不存在会直接报错。
    2. 读取数据
      以上read方法若读到文件末尾，则返回-1。
    3. 释放资源(关闭通道)

字符集

字符流

字符流的底层为字节流。字符流=字节流+字符集。

  • 特点
    1. 输入流：一次读一个字节，遇到中文时，一次读多个字节。
    2. 输出流：底层会把数据按照指定的编码方式进行编码，变成字节再写到文件中。
